## Como este repositório está organizado

Cada estudo de caso segue a mesma estrutura, para facilitar comparação:

1. **Contexto** — o que estava em jogo, quem eram os afetados
2. **Diagnóstico** — como o problema foi identificado e por quê
3. **Decisão** — o trade-off avaliado e a escolha feita (e por quê)
4. **Resultado** — o que mudou, métrica ou impacto observável
